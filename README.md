# IB_HEX_BOT

Bot WhatsApp en français propulsé par **Baileys**.

## Commandes principales avec préfixe `Ib`

- `Ib menu` : Affiche le menu
- `Ib alive` : Vérifie si le bot est en ligne
- `Ib owner` : Montre le propriétaire
- `Ib ping` : Teste la rapidité du bot

## Instructions

1. Clone ce dépôt
2. Installe les dépendances : `npm install`
3. Lance le bot : `npm start`
4. Scanne le QR code pour connecter ton WhatsApp
